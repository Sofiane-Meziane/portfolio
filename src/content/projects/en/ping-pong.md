## Summary
A simple and responsive 2D ping pong game built in C with Raylib. Includes keyboard controls, collision handling, and a basic AI opponent.

## Problem
Build a complete game loop with graphics rendering, input handling, and real-time gameplay.

## Approach
- Raylib for 2D rendering and event handling.
- Procedural game loop with collision detection and score tracking.
- AI paddle that follows the ball trajectory.

## Results
A playable pong game with dynamic ball speed, scoring, and smooth controls.
